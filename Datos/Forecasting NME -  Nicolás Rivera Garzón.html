1. Objetivo ¶

Proyectar las exportaciones no minero-energéticas (NME) para el cierre de 2023 y 2024.

2. Metodología ¶

Se realizará el siguiente proceso:

  1. Importación de datos desde un directorio que se puede cambiar fácilmente para replicar datos.
  2. Análisis exploratorio de datos para encontrar puntos relevantes y características de la serie de tiempo como: estacionalidad y autocorrelación.
  3. Construcción de modelos ARIMA, SARIMA, ETS, Prophet y XGBoost.
  4. Elección del mejor modelo en el conjunto de validación a partir de las métricas de MAE, MSE y RMSE.
  5. Obtención del pronóstico del interés con el modelo elegido.
